# Your Dues Project

A full stack no dues / leave management system.
 [![Watch Demo](https://youtu.be/6yLlMKy7js0)](https://youtu.be/6yLlMKy7js0)
## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB

## Features
- Student dashboard
- Admin panel
- Department approval
- No dues system
