# Your Dues Project

A full stack no dues / leave management system.
 [![Watch Demo](https://img.youtube.com/vi/6yLIMKy7js0/0.jpg)](https://youtu.be/6yLIMKy7js0)
## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB

## Features
- Student dashboard
- Admin panel
- Department approval
- No dues system
